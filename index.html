O que é uma variável em programação?

É um espaço na memória usado para armazenar valores.

O que é um loop?

É uma estrutura que repete instruções enquanto uma condição for verdadeira.

O que é uma função?

Um bloco de código que executa uma tarefa específica e pode ser reutilizado.
